# HeroEngine2

Extension of the HeroEngine project (PP7) with Razor Pages and file persistence.

## Structure

- **HeroEngine.Core** – class library containing game models, abilities, combat, and file services.
- **HeroEngine.Web** – ASP.NET Core Razor Pages web app.

## Requirements

- Visual Studio 2022 with a PC or something can run Visual

## How to run

1. Open the solution `HeroEngine.slnx` in Visual Studio.
2. Set **HeroEngine.Web** as startup project (right‑click → Set as Startup Project).
4. Press F5 or run from terminal:

If that doesn't work, create a new project, right-click on the solution, then add an existing project, and locate HeroEngine.csproj and HeroEngine.Web.csproj.
