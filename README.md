# --- HeroEngine ---

## 📌 Project Description

HeroEngine is a simple C# project that simulates a turn-based RPG combat system.

In this game, heroes fight against enemies using basic attacks and special abilities. Each character has health, level, and different stats that affect the battle.
per turn game

The main goal of this project is to practice object-oriented programming (OOP) 
---

## 🎮 How the game works

The game is based on turn-based combat:

- Heroes attack enemies
- Enemies attack heroes
- Each turn reduces health
- The battle ends when one side reaches 0 health

Some heroes also have special abilities that can:
- Deal extra damage
- Heal allies
- Increase defense
- Control enemies
